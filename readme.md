<strong>Casting</strong> é a modificação do tipo de variável

<br/>

Tipos de variáveis:

- Texto = string()

Operadores aritméticos em python:

- Soma (" + ")
- Subtração (" - ")
- Multiplicação (" \* ")
- Divisão (" / ")
- Divisão Inteira (" // ")
- Modulo (" % ")
- Exponenciação (" \*\* ")
